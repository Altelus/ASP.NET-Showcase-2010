F.A.Qs

  1. How did you create this website?
    • I used notepad to edit the html code, and opened the webpage
      using firefox to test it out. I would then transfer the website files to
      to Zenit using SSH File Transfer, where I would attempt to validate the page
      and correct any errors.
  2. How long did it take you to complete this website?
    • It took about a week, working on it for a few hours a day.
  3. Why have you selected this particular theme?
    • This theme is meant to resemble AlienGUIse's Xenomorph theme on my desktop computer.
  4. How did you create the buttons?
    • I used GIMP (GNU Image Manipulation Program) 2.6.4; a graphics editor
      similar to Adobe Photoshop
  5. How did you get the buttons to change color?
    • I have 2 variant of each button one blue and one yellow, I simply change the
      image depending on the page.
